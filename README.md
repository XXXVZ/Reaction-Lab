## Заказчик
Учебный отдел IT-Куб

## Описание
В IT-Кубе проводят короткие “разминки” перед занятиями, чтобы прокачивать внимание и скорость реакции. Вам поручили сделать MVP консольного тренажёра, который можно запускать на любом школьном ПК без интернета.

## Цель
Сделать программу-тренажёр, которая проводит серию раундов на реакцию, считает время, ошибки, выдаёт статистику и поддерживает режим турнира.

## Критерии готовности
Программа считается готовой, если:
* есть меню (быстрая игра / турнир / правила / выход)
* один раунд: случайная задержка 1–4 сек → команда → ввод → замер времени → проверка → штраф за ошибку
* есть статистика: среднее, лучшее, ошибки
* турнир на 2–4 игроков, определение победителя по правилам сортировки
* код разбит на функции, читаемый, без повторов

## Полное ТЗ
1. **Меню программы**
При запуске показывается меню:
    * Быстрая игра (1 игрок, 5 раундов)
    * Турнир (2-4 игрока, по 5 раундов каждому)
    * Правила
    * Выход
 
Программа работает в цикле и возвращается в меню после игры.

2. **Механика одного раунда (функция `play_round`)**
В одном раунде:
      * Программа печатает: `Готовься...`
      * Ждёт случайное время от 1.0 до 4.0 секунд (`time.sleep(...)`)
      * **Выводит случайную команду одного из типов:**
            * НАЖМИ: А (тут может быть любой символ)
            * ВВЕДИ: go (тут может быть любая строка)
      * **Засекает время реакции**:
            * `start = ...`
            * `end = ...`
            * `end - start`
      * **Проверяем ответ**:
            * если ответ не совпал, тогда начисляется штраф: **+2.0 секунды**, и раунд считается **ошибочным**
      * **Возвращает результат раунда (например: время с учётом штрафа, была ли ошибка, правильная команда)**

3. **Игра для одного игрока** (функция `play_game`)
      * На вход: `player_name`, `rounds`
      * Запускает `rounds` раундов
      * Собирает результаты
      * В конце выводит статистику:
            * среднее время
            * оучгее время
            * количество ошибок

4. **Турнир** (функция `tournament`)
      * Запросить количество игроков 2–4
      * Запросить имена игроков
      * Каждый играет по 5 раундов
      * Сохранить итог каждого игрока (среднее, ошибки, лучшее)
      * Победитель:
            * меньше среднее время
            * если среднее одинаковое, значит выиграл тот, у кого **меньше ошибок**
            * если и это одинаковое, тогда выиграл тот, у кого лучшее время

## Каркас кода
Вот какие функции могут быть:
* `print_menu()`
* `show_rules()`
* `get_command()` — выбирает случайную команду (типа НАЖМИ, или ВВЕДИ)
* `play_round(penalty=2.0)` — один раунд (`penalty` - время, добавляемое за каждую ошибку в ответе)
* `calc_stats(results)` — считает среднее/лучшее/ошибки
* `print_player_summary(name, stats)` — красиво печатает итоги
* `play_game(player_name, rounds=5)` — игра игрока
* `tournament(rounds=5)` — турнир
